Oman Feedback Platform

Welcome to the Oman Feedback Platform

This platform allows citizens to submit feedback, suggestions, and complaints to local authorities, and enables authorities to respond and manage these submissions.

API Status

✅ API is operational

The frontend application is currently under maintenance.

API Endpoints

  • GET /api - API Root
  • POST /api/auth/login - User Login
  • POST /api/auth/register - User Registration
  • GET /api/feedback - Get Feedback Submissions
  • POST /api/feedback - Submit New Feedback
